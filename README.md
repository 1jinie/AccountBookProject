# Simple한 가계부
콘솔 환경에서 동작하는 java기반의 가계부 프로그램입니다.  
사용자는 수입/지출을 기록하고, 월별 내역과 잔액을 확인할 수 있습니다.  
입출력/자료구조/객체지향 설계를 연습합니다.

## Tech stack
- java (Zulu 21.0.11)
- 파일 입출력(File I/O)
- 컬렉션 프레임워크(ArrayList, HashMap)

## 필드
- int id : 고유 식별자
- String type : 수입/지출
- int amount : 금액
- String description : 내용
- String category : 분류(식비, 교통, ...)
- Calendar date : 날짜

## 주요 기능
- 수입 / 지출 입력
- 전체 내역 조회
- 월별 내역 조회
- 잔액 계산
- 데이터 파일 저장 및 불러오기
